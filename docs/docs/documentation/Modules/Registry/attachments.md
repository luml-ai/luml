# Attachments

Attachments are files saved alongside a run in the Registry. They appear on the Attachments tab of both models and experiments and capture outputs produced during training and evaluation. The module imposes no restrictions on the type or extension of attached files. These can range from model weights to configuration files, audio samples, or project documentation.

For seamless interaction, a File Viewer is integrated into the Registry interface. It enables in-browser preview of commonly used file formats without requiring downloads. Supported formats include media content such as images (.jpeg, .png, .svg), audio recordings (.mp3), and video files (.mp4). The viewer also handles data and structural files including tabular data (.csv), configuration files, and scripts (.json, .xml, .txt), as well as documents such as PDF reports (.pdf) and web pages (.html).

Files in other formats (e.g., binary data or proprietary extensions) or files over 15 MB remain available for download to a local device.
